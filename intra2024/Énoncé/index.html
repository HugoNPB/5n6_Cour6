QUESTION 2 - Accès réseau

Tu dois créer un projet appelé nom_web_service

Web service:

Ton code doit accéder à un service web disponible en méthode GET sur l'URL ci-dessous. 

http://localhost:8080/exam/animaux/{continent}?famille={famille}
Exemple concret 
http://localhost:8080/exam/animaux/afrique?famille=felidae
Le paramètre famille est optionnel et on peut récupérer tous les animaux pour l'ensemble de la planète avec le "continent" "monde" 
http://localhost:8080/exam/animaux/monde
ou encore 
http://localhost:8080/exam/animaux/monde?famille=

Les paramètres continent et famille sont entrés via 2 champs texte.

Le service retourne une liste d'animaux trouvés selon le continent et la famille de l'animal.

Les données retournées par le service doivent être affichées dans un ListView. Le nom de l'animal doit être visible. Si le continent de l'animal est Océans, on affiche l'icône "water", sinon on affiche l'icône "landscape".

Icon(Icons.water) et Icon(Icons.landscape)

L'interface n'a pas à être identique à cette affichée en démo, elle doit seulement contenir les mêmes éléments.

Pointage

1 points Ton application utilise 2 champs texte pour récupérer le continent et la famille des animaux. Si tu n'y arrives pas tu peux envoyer 2 constantes mais tu perds ce point.

2 points Récupérer les données retournées par le service dans l'application et les afficher via un print ou tout autre moyen.

2 points Récupérer les données retournées par le service dans l'application et les afficher via un ListView où on voit tous les champs de données. Si tu as réussi à afficher un ListView tu obtiens également les 2 points précédents.
